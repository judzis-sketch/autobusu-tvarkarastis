# **App Name**: eTransport PWA

## Core Features:

- Route Display: Display available bus routes with their numbers and names.
- Timetable Display: Show the timetable for a selected route, including stop names and scheduled times.
- Map Visualization: Display bus stops on a map using Leaflet, with popups showing stop names and times.
- Admin Route Creation: Allow administrators to create new bus routes by entering a number and a name.
- Admin Timetable Entry: Enable administrators to add timetable entries for a selected route, including stop names, times, and coordinates.
- Data Storage: Store route and timetable data in Firestore.

## Style Guidelines:

- Primary color: Dark Blue (#1E3A8A), conveying trust and reliability for a transportation app.
- Background color: Light Gray (#F9FAFA), providing a clean and neutral backdrop for content.
- Accent color: Teal (#2DD4BF), used for interactive elements and highlights, offering a modern, digital touch.
- Body and headline font: 'PT Sans', a humanist sans-serif combining modern look and warmth, for headlines or body text.
- Use simple, consistent icons to represent bus stops and routes on the map and in the timetable display.
- Maintain a clean and organized layout with clear distinctions between sections. Utilize a responsive design to ensure optimal viewing on various devices.
- Implement subtle transitions and animations for loading data and displaying new information.
